# Job Scraper

This project scrapes job listings from a website and saves them into CSV and Excel formats.

## Features
- Extracts job title, company, location, and apply link
- Supports keyword filtering
- Saves data to CSV and Excel

## Technologies
- Python
- Requests
- BeautifulSoup
- Pandas

## How to Run
pip install -r requirements.txt
python src/scraper.py

## Output
- data/jobs.csv
- data/jobs.xlsx
